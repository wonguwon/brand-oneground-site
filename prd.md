# [PRD] 원그라운드(One Ground) × 스페이스데스크 — 브랜드 & AI CRM SaaS 웹사이트 구축

## 1. 프로젝트 개요 (Project Overview)

* **브랜드:** 원그라운드 (One Ground) — AI 솔루션 개발 브랜드
* **제품명:** 스페이스데스크 — 무인 공간 전용 AI CRM SaaS
* **제품 의미:** "밤새 공간을 지켜준다" — 운영자가 자리를 비워도 AI가 문의·예약·알림을 24시간 처리
* **슬로건:** "스페이스데스크가 지키는 동안, 당신은 쉬세요."
* **핵심 비전:** 무인 공간 운영자의 반복 업무(고객 문의 응대 · 예약 확인 문자 발송 · 매출 집계)를 스페이스데스크 하나로 완전 자동화. 직접 파티룸을 운영하며 검증한 실전 솔루션임을 강조.
* **주요 목표:**
  1. **제품 신뢰도:** "직접 운영하며 만든 솔루션"이라는 실증 스토리로 리드 전환율 극대화.
  2. **전문성 신호:** 10년 개발 경력 · 다수 강의 이력으로 기술력과 교육 역량을 동시에 증명.
  3. **리드 확보:** 스페이스데스크 사전 도입 신청 100건 이상 확보.

## 2. 스페이스데스크 핵심 기능 (Core Features)

### 2.1. 🤖 RAG 기반 문의응답 AI 챗봇
운영자의 FAQ·가격표·예약 정책을 학습해 24시간 고객 문의에 자동 답변하는 스페이스데스크의 핵심 엔진.

| 항목 | 내용 |
|---|---|
| **기술 방식** | 운영자의 FAQ·가격표·예약 정책 문서를 벡터 DB에 인덱싱 후 LLM으로 정확한 답변 생성 (RAG 아키텍처) |
| **주요 채널** | 카카오채널 / 인스타 DM / 자체 웹 위젯 (멀티채널 지원) |
| **핵심 효과** | 24시간 무인 응대 · 반복 문의 85% 자동 처리 · 운영자 야간 부재 시에도 예약 기회 미손실 |
| **차별점** | 일반 챗봇과 달리 운영자 문서를 학습하므로 "가격이 얼마예요?", "주차 가능해요?" 같은 실제 문의에 정확히 답변 |

### 2.2. 📱 자동 예약 문자
예약 생성부터 완료까지 전 주기를 자동화하는 알림톡·SMS 발송 모듈.

| 항목 | 내용 |
|---|---|
| **트리거 방식** | 예약 생성·변경·D-1·당일·완료 이벤트에 따른 자동 발송 |
| **발송 채널** | 알림톡(카카오) / SMS / LMS |
| **핵심 효과** | 노쇼(No-show) 감소 · 운영자 수동 문자 발송 시간 제거 · 리뷰 유도로 신규 고객 전환 촉진 |
| **연동** | 네이버 예약 / 직접 예약 폼 / 카카오맵 예약 연동 지원 (로드맵) |

### 2.3. 📅 예약 관리 & 고객 관리 (CRM)
예약 현황을 한눈에 파악하고, 고객 데이터를 체계적으로 관리하는 통합 CRM 대시보드.

| 항목 | 내용 |
|---|---|
| **예약 관리** | 달력/리스트 뷰 예약 현황 조회 · 예약 생성·수정·취소 · 잔금 확인 상태 관리 |
| **고객 관리** | 고객별 예약 이력 · 재방문율 · 선호 룸/시간대 분석 |
| **노쇼 방지** | 예약 미확인 고객 자동 알림 발송 · 블랙리스트 관리 |
| **운영 리포트** | 일/주/월별 예약 건수, 매출, 객단가 대시보드 자동 집계 |

### 2.4. 💰 매출 관리
결제 수단별 수익을 자동 집계하고, 운영 현황을 실시간으로 파악하는 매출 관리 모듈.

| 항목 | 내용 |
|---|---|
| **결제 현황** | 현금·카드·계좌이체·간편결제(카카오페이 등) 수단별 매출 통합 조회 |
| **미수금 관리** | 잔금 미납 예약 자동 감지 · 잔금 요청 알림 발송 |
| **정산 리포트** | 일/주/월/연간 매출 합계 · 룸별·시간대별 수익 분석 자동 생성 |
| **세금계산서** | 사업자 고객 세금계산서 발행 이력 관리 (로드맵) |

## 3. 신뢰도 구성 요소 (Trust Signals)

### 3.1. 🎮 오프라인 운영 — 게임파티룸 밤샘
* 직접 파티룸(게임파티룸 밤샘)을 운영하며 고객 문의·예약·노쇼 문제를 직접 체감.
* 스페이스데스크를 자사 파티룸에 먼저 적용 → 실제 운영 데이터를 바탕으로 제품을 지속 개선.
* **스토리 포인트:** "스페이스데스크는 제가 직접 운영하는 파티룸에서 먼저 검증했습니다."

### 3.2. 👨‍💻 10년 개발 경력 & 강의 이력
* 10년차 풀스택/AI 개발자로서 제품의 기술적 완성도 보장.
* 기업 DX 강의, KDT 부트캠프 등 다수의 강의를 진행한 교육자로서 실무 중심 솔루션 설계.
* **스토리 포인트:** "기술과 현장, 교육을 모두 아는 사람이 만든 솔루션입니다."
